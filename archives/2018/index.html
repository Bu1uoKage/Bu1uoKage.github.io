PHP核心配置

了解php的配置对web渗透还是很重要的~

基础语法

  1. 大小写敏感
    directive = value (指令=值)
    foo=bar ≠ FOO=bar
  2. 运算符
    |、&、~、!
  3. 空值的表达式
    foo = ;
    foo = none ;
    foo = “none” 是空的字符串

安全模式

safe_mode = off
安全模式，用来限制文档的存取
